# CRM Module

## Purpose
Manage all client interactions, leads, and requests.

## Features
- Identify caller by membership ID or phone
- Store client profiles (preferences, history)
- Handle inbound requests (calls, web, WhatsApp)
- Allow sales team to search flights on behalf of clients

## Workflow
1. Client contacts sales
2. Sales identifies client (member / non-member)
3. Request is created
4. Linked to booking module

## Integrations
- Inventory module (search aircraft)
- Booking module (create quote)
- Membership module (validate status)
