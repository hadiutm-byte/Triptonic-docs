# Universal Jets — Quote Flow

## 1. Purpose

Define how a sales agent creates and sends a charter quote to a client.

---

## 2. Trigger

Quote starts when:

- client calls
- client submits request online
- existing client requests new trip

---

## 3. Step-by-Step Flow

### Step 1: Identify Client (CRM)
- search client by name / phone
- or create new client

---

### Step 2: Create Request
- departure / arrival
- dates
- passengers
- preferences (smoking, pets, catering)

---

### Step 3: Search Inventory
- find available aircraft
- filter by category, location, budget

---

### Step 4: Pricing
- calculate flight cost
- include positioning
- include extras
- apply margin

---

### Step 5: Select Option
- choose best aircraft
- validate availability with operator

---

### Step 6: Generate Quote
- aircraft details
- price breakdown
- total price
- validity time

---

### Step 7: Send to Client
- email / WhatsApp
- branded format

---

## 4. Status Tracking

Each quote must have status:

- draft
- sent
- pending
- accepted
- rejected

---

## 5. Conversion

If client accepts:

→ move to booking  
→ generate contract  
→ trigger payment  

---

## 6. KPIs

- quote to booking ratio
- response time
- average margin per deal
